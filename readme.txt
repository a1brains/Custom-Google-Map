Simplest and Easiest way to display Google Maps in Website Pages & its Blog Posts in seconds.

== Description ==

Add a customized map quickly and easily in seconds using Latitude and Longitude of any address; simply add the generated shortcode to display it anywhere in WordPress website pages, blog posts, etc.

Below is the list of feature

1) Genarete your google map in a minuit of time
2) You can create as many as map you want
3) You can use genrated shortcode anywhere on Post / Page /  Custome post
4) Provide Option to use custome Map Marker
5) Provide map customise option like, Enable or disable scroll wheel, double click zoom, Zoom control etc

== Frequently Asked Questions ==

= How to find Latitude and Longitude of any address?  =

Follow below mentioned steps

- On your computer, visit Google Maps.

- Right-click a location on the map.

- Select “What&#39;s here?”

- A card appears at the bottom of the screen with Latitude &amp; Longitude info.

NOTE: Alternatively Latitude &amp; Longitude info. is also available in URL (Hint: For an

already identified address you can directly look for Latitude &amp; Longitude info. from URL.

Further, first number is always for Latitude and second Longitude. For instance,

https://www.google.co.in/maps/place/A1brains+Infotech/@23.0295322,72.5252264,17z/data

= How to create custom map and use it my Website Page or Post? =

1)goto Maps->Add New
2)Fill out all filed and click "Publish" button


= How to use map in mysite =

Step 1: Install the plugin (Refer section “Installing Plugins” -https://codex.wordpress.org/Managing_Plugins)

Step 2: Click on &quot;Maps&quot; in your left navigation menu in WP Admin Dashboard.

Step 3: Click “Add New” and Start customizing your Google Map to display it as per your

preferences:

- Choose the appropriate Name for your Map

- Put Latitude and Longitude (See how to find Latitude and Longitude of any address in

FAQs)

- Choose Image for Marker

- Set Height &amp; Zoom Level

- Enable / Disable to choose whether or not to keep your map Draggable

- Enable / Disable to choose whether or not to keep your map Scrollable

- Enable / Disable to choose whether or not to have Zoom Control

- Choose to keep Double Click Zoom enabled or disabled

- Hit the Update button at the top right.

Step 4: Completing above process will give you shortcode to use it in the website pages or

blog posts to display Map.
